# HandyWote Portfolio

这是我的 3D 作品集网站，基于 Three.js 构建，展示了一个交互式的 3D 场景。

## 在线演示

[https://handywote.top](https://handywote.top)

## 技术栈

- **3D 引擎**: Three.js + CSS3DRenderer + GLTFLoader
- **构建工具**: Vite
- **语言**: TypeScript

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm start
```
